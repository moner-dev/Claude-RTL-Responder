إضافة متكاملة لمتصفح Firefox تُحسِّن تجربة استخدام Claude.ai باللغة العربية. تكتشف الأداة النصوص العربية تلقائياً وتُطبِّق عليها اتجاه RTL الصحيح، مع الحفاظ على اتجاه LTR لكتل الكود والمحتوى التقني. تدعم الإضافة المحتوى المختلط، التخصيص الكامل عبر صفحة الإعدادات، واختصارات لوحة المفاتيح لتبديل سريع.

A comprehensive Firefox extension that enhances Claude.ai for Arabic users. It automatically detects Arabic text and applies proper RTL direction while preserving LTR for code blocks and technical content. Supports mixed-language content, full customization via the settings page, and keyboard shortcuts for quick mode switching.

المميزات الرئيسية Key Features

  • كشف تلقائي للنصوص العربية Automatic RTL detection
  • معالجة ذكية للنصوص المختلطة Mixed-content handling
  • الحفاظ على كتل الكود كـ LTR Code block preservation
  • تخصيص حجم الخط Font size customization
  • تخصيص تباعد الأسطر Line spacing customization
  • مؤشر بصري اختياري RTL indicator
  • اختصارات لوحة المفاتيح Keyboard shortcuts
  • دعم الوضع الداكن Dark mode support
  • واجهة ثنائية اللغة Bilingual UI
