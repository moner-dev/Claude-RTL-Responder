## 🔒 Security Features | ميزات الأمان

### Data Handling | التعامل مع البيانات

- ✅ **No Data Collection** — No data is sent to external servers
- ✅ **Local Storage Only** — All settings are stored in your browser
- ✅ **No Tracking** — We don't use any analytics or tracking tools
- ✅ **Open Source** — You can review the entire codebase

### Code Security | أمان الكود

- ✅ No remote code execution
- ✅ No eval() or dynamic code injection
- ✅ No external script loading
- ✅ Content Security Policy compliant
- ✅ Manifest V3 architecture
