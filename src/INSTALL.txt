[ENGLISH]

FIREFOX:
1. Extract this ZIP to a permanent folder
2. Open Firefox → about:debugging#/runtime/this-firefox
3. Click "Load Temporary Add-on..."
4. Select "manifest.json" from extracted folder
5. Visit https://claude.ai — Done!

CHROME:
1. Extract this ZIP to a permanent folder
2. Open Chrome → chrome://extensions
3. Enable "Developer mode" (top right toggle)
4. Click "Load unpacked"
5. Select the extracted folder
6. Visit https://claude.ai — Done!

Shortcuts: Alt+G (toggle RTL) | Alt+O (options)

Note: Temporary extensions are removed on browser restart.

Full guide: https://github.com/moner-dev/Claude-RTL-Responder
